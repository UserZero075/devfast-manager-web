Montaje en minutos

Instala backend, genera Prisma, levanta la API y conecta el frontend con VITE_API_URL.

git clone https://github.com/UserZero075/Business-Management-Dashboard.git
cd Business-Management-Dashboard/backend
npm install && npx prisma generate && npx prisma db push && npm run dev

cd ../frontend
npm install
VITE_API_URL=http://localhost:3001 npm run dev
